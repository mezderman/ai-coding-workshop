# Habit Tracker

The app for the Virgo Group AI Coding Workshop (see the [repo README](../README.md)
for the full workshop flow).

Single user, no auth. Create habits, check in once per day, see them in a journal.
Next.js + TypeScript + SQLite. This is the **finished** version — the streaks +
stats "motivation layer" described in `client-brief.md` is already built. See
`prd/` for the PRDs and `issues/done/` for the issues that produced it.

## Run

```bash
npm install
npm run dev
```

## Test / typecheck

```bash
npm run test
npm run typecheck
```
